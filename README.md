# e-plantShopping

e-plantShopping is a web-based plant shopping application that allows users to explore and purchase a variety of plants online. This project demonstrates modern front-end development practices using React and component-based architecture.

## Project Overview

The e-plantShopping application provides users with a clean and interactive interface to:

- Browse different types of plants
- View plant details
- Add plants to the shopping cart
- Manage cart items
- Simulate a simple e-commerce experience

## Features

- Component-based UI structure
- Dynamic rendering using React
- State management for cart functionality
- Responsive and user-friendly design
- Organized and modular code structure

## Technologies Used

- React
- JavaScript (ES6+)
- HTML5
- CSS3
